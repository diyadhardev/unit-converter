# Unit Converter

A simple and intuitive web-based unit converter application that allows you to convert between different units of measurement.

**Project Source:** [roadmap.sh/projects/unit-converter](https://roadmap.sh/projects/unit-converter)

## Features

- 🔄 **Multiple Unit Categories**
  - **Length**: Kilometers, Centimeters, Meters, Millimeters, Miles, Yards, Feet, Inches
  - **Weight**: Kilograms, Grams, Milligrams, Pounds, Ounces, Tonnes
  - **Temperature**: Celsius, Fahrenheit, Kelvin

- 💻 **User-Friendly Interface**: Clean and intuitive design for easy navigation
- ⚡ **Real-Time Conversion**: Instant conversion results
- 🎯 **Tab-Based Categories**: Switch between different unit types with ease
- 🔧 **Reset Functionality**: Clear results and start a new conversion

## Technologies Used

- **HTML5**: Markup structure
- **CSS3**: Styling and layout
- **JavaScript (Vanilla)**: Core functionality and logic

## How to Use

1. **Open the Application**: Open `index.html` in your web browser
2. **Select a Category**: Click on a tab (Length, Weight, or Temperature) to choose the type of unit
3. **Enter Value**: Input the value you want to convert
4. **Select Units**: Choose the unit to convert from and the unit to convert to
5. **Convert**: Click the "Convert" button to see the result
6. **Reset**: Click "Reset" to clear the form and results

## Installation

No installation is required! Simply:

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start converting units

## Usage Example

To convert 5 meters to feet:

1. Select the "Length" tab
2. Enter "5" in the input field
3. Select "Meters(m)" as the unit to convert from
4. Select "Feet(ft)" as the unit to convert to
5. Click "Convert"
6. Result: "5 Meters(m) = 16 Feet(ft)"
